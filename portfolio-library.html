LITTLE LIBRARIES

A short subtitle

Little Chicago Library is an application developed to incentivise reading, and impact the community of Chicago by showing the Little Libraries available in the borough of Ravenswood, and the books available at each library.

Built with the community in mind Little Libraries hopes to facilitate readers to know which books are available at each location, if a book has been adopted, or simply checked out. You also can donate books, and add a new little library location for other readers to see, if you find one that isn't in our database yet, or if you set one up yourself.

Little Chicago Library uses the Google maps API and Google Geolocation API to render a map with the Little Libraries in the Ravenswood area, and to have the ability to add new libraries. I built a Ruby on Rails backend API to create an inventory system of available books at each library. This application was developed in Vanilla JavaScript which allows complex DOM manipulation.
