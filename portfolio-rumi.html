RUMI

An Agile House

Rumi is a mobile application developed for both IOS and Android devices that is based on Agile points. House chores are divided amongst the "Rumis" living in the same household.

Rumi is built to randomly assign the household chores between "rumis" as evenly distributed as possible with the help of recursion for a sprint. Once the chores have been assigned the "rumis" must either confirm or reject the sprint, a consensus of over 50% must be reached. Once accepted the sprint is now live and every rumi can view all the chores in the home feed. If rejected a new sprint is created until consensus is reached.

Using technologies such as Expo, Ruby on Rails, PostgreSQL, React Native, and Redux for state management, Rumi is a robust and scalable application which uses its own API and database to store user and household information.
