FLATNOTE

Secure notes

Flatnote is an application built for users to create and save notes in a secure way. To use this application users must sign up either by creating a username and password or by using Google sign up / login.

Once logged in users can either create, edit, and delete notes that they decide to write

Flatnote is developed with React, and utilizes Redux for state management. By implementing JSON Web Tokens and localStorage we can store encrypted user information client-side and assure a secure user login. Another method employed for security was Google OAuth for a secure login / signup and authentication. This application uses a Ruby on Rails backend API endpoint to and PostgreSQL database to store user information and notes which are created.
